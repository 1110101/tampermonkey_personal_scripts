Overview

What is Tampermonkey?

Enhance any website with custom scripts, no server changes required.

Browser Extension

A free extension for Chrome, Firefox, Safari, Edge, and Opera that lets you run custom JavaScript on any website.

Customize Websites

Modify behavior, add features, automate tasks, and improve UI without touching the original website.

Safe & Controlled

Scripts run in a sandboxed environment with explicit permissions. Enable or disable any script at any time.

Setup

How to Install & Use

Get up and running in a few minutes.

1

Install Tampermonkey

  • Chrome: Chrome Web Store
  • Firefox: Firefox Add-ons
  • Edge: Edge Add-ons
  • Safari: Mac App Store
2

Install a UserScript

Click any "Install" link on this page, or navigate to the raw file on GitHub. Tampermonkey detects .user.js files automatically.

1. Click "Install" on a script card below
2. Tampermonkey opens and detects the script
3. Click "Install" to confirm
3

Enable & Configure

  • Scripts are enabled by default on matching sites
  • Click the Tampermonkey icon to see active scripts on the current page
  • Toggle scripts on/off individually
  • Edit scripts anytime via Dashboard
4

Auto-Updates

All scripts include @updateURL and @downloadURL metadata and update automatically.

  • Tampermonkey checks periodically in the background
  • Updates require confirmation before installing
  • No manual reinstallation needed
