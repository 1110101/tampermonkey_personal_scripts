What is Tampermonkey?

Enhance your browsing experience with custom scripts

Browser Extension

Tampermonkey is a free browser extension available for Chrome, Firefox, Safari, Edge, and Opera that lets you run custom JavaScript code (UserScripts) on any website.

Customize Websites

UserScripts can modify webpage behavior, add new features, automate tasks, improve UI/UX, and integrate with external APIs - all without changing the original website.

Safe & Controlled

Scripts run in a sandboxed environment with explicit permissions. You control which scripts run on which sites. Scripts are stored locally and can be enabled/disabled anytime.

How to Install & Use

Get up and running in just a few minutes

1

Install Tampermonkey Extension

First, install the Tampermonkey browser extension:

  • Chrome: Chrome Web Store
  • Firefox: Firefox Add-ons
  • Edge: Edge Add-ons
  • Safari: Mac App Store
2

Install UserScripts

Easy Installation: Click the "Raw" button on any .user.js file on GitHub

1. Browse to GitHub Repository
2. Click on a .user.js file
3. Click the "Raw" button
4. Tampermonkey will auto-detect it
5. Click "Install"

Scripts with @updateURL will auto-update!

3

Enable & Configure

After installation:

  • Scripts are enabled by default on matching websites
  • Click the Tampermonkey icon to see active scripts on current page
